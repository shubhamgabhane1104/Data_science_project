# 🏡 Bangalore House Price Prediction  
### End-to-End Machine Learning + Interactive Web App

---

##  Project Overview
This project is a **full-stack machine learning application** that predicts house prices in Bangalore based on user inputs like square footage, BHK, bathrooms, balcony, and location.

It goes beyond model building by delivering a **clean, interactive user interface** for real-time predictions — making it closer to a production-ready system.

---

##  What Makes This Project Stand Out
- ✅ Real-world dataset with heavy preprocessing  
- ✅ Feature engineering & outlier handling  
- ✅ Trained ML model with saved artifacts  
- ✅ Interactive UI for real-time predictions  
- ✅ End-to-end pipeline (data → model → app)

## ⚙️ Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **Visualization:** Matplotlib, Seaborn  
- **Backend:** Streamlit  

---

##  Data Processing Pipeline
- Cleaned missing & inconsistent data  
- Converted range values (e.g., sqft ranges)  
- Reduced location dimensionality  
- One-hot encoding for categorical variables  
- Removed outliers using domain logic  

---

##  Model Details
- **Algorithm:** Random Forest  
- **Evaluation Metrics:**
  - R² Score  
  - Mean Absolute Error
